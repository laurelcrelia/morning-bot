# morning-bot
Morning Briefing Script

## Pre-requisites
- Python 3.8+

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/morning-bot.git
   cd morning-bot
   ```
2. Activate venv
   ```bash
   source venv/bin/activate
   ```
3. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
4. Run the script
   ```bash
   python src/main.py
   ```
